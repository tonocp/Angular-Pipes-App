Textos y fechas

Pipes incluidos en Angular - Usualmente se encuentran en el Common Module.

Textos

Pipes para usar con variables de tipo string.

Variable Original = {{nombreLower}}
Resultado = {{nombreLower | uppercase}}
Variable Original = {{nombreUpper}}
Resultado = {{nombreUpper | lowercase}}
Variable Original = {{nombreCompleto}}
Resultado = {{nombreCompleto | titlecase}}

Fechas

Pipes para mostrar las variables de tipo Date.

  • Variable fecha date = new(Date) --> fecha
  • Pipe Date --> fecha | date
  • Argumento 'short' --> fecha | date:'short'
  • Argumento 'long' --> fecha | date:'long'
  • Argumento personalizado --> fecha | date:''HH:mm:ss - dd/MM/yyyy - GGGG''
  • Argumento Long y modificar Locale --> fecha | date:'long':'GMT+6'
  • Argumento Long y modificar Locale --> fecha | date:'long':'GMT-6'
  • Argumento Long y modificar Idioma --> fecha | date:'long':' ':'en'
  • Argumento Long y modificar Idioma --> fecha | date:'long':' ':'fr'
  • {{fecha}}
  • {{fecha | date}}
  • {{fecha | date:'short'}}
  • {{fecha | date:'long'}}
  • {{fecha | date:'HH:mm:ss - dd/MM/yyyy - GGGG'}}
  • {{fecha | date:'long':'GMT+6'}}
  • {{fecha | date:'long':'GMT-6'}}
  • {{fecha | date:'long':' ':'en'}}
  • {{fecha | date:'long':' ':'fr'}}
